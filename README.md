# NovaNetX VPN

Windows VPN / proxy client by Netch Solutions — system-wide TUN tunneling for
VLESS / REALITY / Trojan / Shadowsocks / WireGuard and more.

[![Release](https://img.shields.io/github/v/release/ShanudhaTirosh/NovaNetX-VPN?style=flat-square)](https://github.com/ShanudhaTirosh/NovaNetX-VPN/releases)

### Modes
- `ProcessMode` — Netfilter driver intercepts per-process traffic
- `ShareMode` — share your connection via WinPcap / Npcap
- `TunMode` — WinTUN virtual adapter for full system-wide tunneling
- `WebMode` — web proxy mode

### Protocols
- [`Socks5`](https://www.wikiwand.com/en/SOCKS)
- [`Shadowsocks`](https://shadowsocks.org)
- [`ShadowsocksR`](https://github.com/shadowsocksrr/shadowsocksr-libev)
- [`WireGuard`](https://www.wireguard.com)
- [`Trojan`](https://trojan-gfw.github.io/trojan)
- [`VMess`](https://www.v2fly.org)
- [`VLESS`](https://xtls.github.io)

### Other
- UDP NAT FullCone (depends on your server)
- .NET 6.0 x64 (Windows)
